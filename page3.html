JavaScript Methods and Concepts

  • push() - Adds one or more elements to the end of an array.
  • pop() - Removes the last element from an array.
  • shift() - Removes the first element from an array.
  • unshift() - Adds one or more elements to the beginning of an array.
  • concat() - Merges two or more arrays into one.
  • join() - Joins all elements of an array into a string.

  • indexOf() - Returns the first index of a given element in an array.
  • lastIndexOf() - Returns the last index of a given element in an array.
  • includes() - Checks if an array contains a specific element.

  • sort() - Sorts the elements of an array in place.
  • reverse() - Reverses the order of the elements in an array.

  • forEach() - Executes a provided function once for each array element.
  • map() - Creates a new array with the results of calling a function for every array element.
  • filter() - Creates a new array with all elements that pass the test implemented by the provided function.
  • reduce() - Applies a function against an accumulator and each element to reduce it to a single value.

In JavaScript, arrays declared with const cannot be reassigned, but the elements within the array can be modified.

  • new Date() - Returns the current date and time.
  • Date.now() - Returns the number of milliseconds elapsed since January 1, 1970.
  • new Date(milliseconds) - Returns a date object from the specified milliseconds since January 1, 1970.

  • toDateString() - Returns a date as a string (e.g., "Sat Nov 24 2024").
  • toISOString() - Returns a string in ISO format (e.g., "2024-11-24T14:45:00.000Z").
  • toLocaleDateString() - Returns a string representing the date in the local format.

  • getFullYear() - Returns the year of a date.
  • getMonth() - Returns the month (0-11) of a date.
  • getDate() - Returns the day of the month (1-31).
  • getDay() - Returns the day of the week (0-6).
  • getHours() - Returns the hours (0-23).

  • setFullYear() - Sets the year of a date.
  • setMonth() - Sets the month of a date (0-11).
  • setDate() - Sets the day of the month of a date.
  • setHours() - Sets the hours of a date (0-23).

  • Math.max() - Returns the largest number from a list of numbers.
  • Math.min() - Returns the smallest number from a list of numbers.
  • Math.random() - Returns a random number between 0 and 1.

Math.random() - Generates a random floating-point number between 0 (inclusive) and 1 (exclusive). You can use it to create random numbers within any desired range.

Booleans are logical data types that can only be either true or false. They are often used in control flow statements like conditionals and loops.

  • == - Compares two values for equality (without considering data type).
  • === - Compares two values for equality, considering data type.
  • != - Checks if two values are not equal (without considering data type).

The if else statement is used to execute code based on a condition. The structure looks like:

if (condition) {
// code to execute if condition is true
} else {
// code to execute if condition is false
}

The switch statement evaluates an expression and executes the block of code corresponding to the matched case.

switch (expression) {
case value1:
  // code to execute if expression matches value1
  break;
case value2:
  // code to execute if expression matches value2
  break;
default:
  // code to execute if no case matches
}

The for loop is used to execute a block of code a specific number of times. It includes an initialization, condition, and increment expression:

for (let i = 0; i < 5; i++) {
// code to be executed
}
