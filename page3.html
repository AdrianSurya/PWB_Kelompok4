1. JS Array Methods

JavaScript menyediakan banyak metode untuk memanipulasi array, seperti menambah, menghapus, atau menggabungkan elemen.

Beberapa metode umum yang dibahas adalah:

  • push(): Menambahkan elemen ke akhir array.
  • pop(): Menghapus elemen terakhir dari array.
  • shift(): Menghapus elemen pertama dari array.
  • unshift(): Menambahkan elemen ke awal array.
  • concat(): Menggabungkan dua array.
  • slice(): Mengambil bagian dari array tanpa mengubah array asli.
  • splice(): Mengubah array dengan menambahkan atau menghapus elemen.
  • forEach(): Menjalankan fungsi untuk setiap elemen dalam array.
  • map(): Membuat array baru dengan hasil dari fungsi yang dijalankan pada setiap elemen.
  • filter(): Membuat array baru dengan elemen yang memenuhi kondisi tertentu.
  • reduce(): Mengurangi array menjadi satu nilai dengan menjalankan fungsi pada setiap elemen.
Contoh:

let arr = [1, 2, 3];
arr.push(4);  // Menambahkan elemen 4 di akhir array
console.log(arr);  // Output: [1, 2, 3, 4]

arr.pop();  // Menghapus elemen terakhir
console.log(arr);  // Output: [1, 2, 3]

2. JS Arrays Search

Metode pencarian dalam array yang ada di W3Schools antara lain:

  • indexOf(): Menemukan indeks pertama dari elemen yang dicari.
  • includes(): Mengecek apakah suatu elemen ada di dalam array.
  • find(): Menemukan elemen pertama yang memenuhi kondisi.
  • findIndex(): Menemukan indeks pertama dari elemen yang memenuhi kondisi.
Contoh:

let arr = [10, 20, 30];
let index = arr.indexOf(20);  // Menemukan indeks 20
console.log(index);  // Output: 1

let exists = arr.includes(30);  // Mengecek apakah 30 ada di array
console.log(exists);  // Output: true

3. JS Array Sort

W3Schools menjelaskan bagaimana cara menyortir array:

  • sort(): Menyortir array. Secara default, sort() menyortir elemen sebagai string.
  • sort() dengan fungsi kustom: Untuk mengurutkan angka dengan benar, gunakan fungsi pembanding.
Contoh:

let arr = [3, 1, 2];
arr.sort();  // Menyortir array sebagai string
console.log(arr);  // Output: [1, 2, 3]

arr.sort((a, b) => a - b);  // Mengurutkan secara numerik
console.log(arr);  // Output: [1, 2, 3]

4. JS Array Iteration

W3Schools juga mengajarkan cara iterasi array menggunakan berbagai metode:

  • for loop: Loop standar untuk mengakses setiap elemen array.
  • forEach(): Metode untuk iterasi setiap elemen dengan menjalankan fungsi tertentu.
Contoh:

let arr = [1, 2, 3];
arr.forEach(function(item) {
console.log(item);  // Output: 1, 2, 3
});

for (let i = 0; i < arr.length; i++) {
console.log(arr[i]);  // Output: 1, 2, 3
}

5. JS Array Const

Di W3Schools, dijelaskan bahwa meskipun Anda dapat mendeklarasikan array menggunakan const, Anda masih bisa mengubah isi array (seperti menambah atau menghapus elemen), tetapi Anda tidak bisa mengubah referensi array itu sendiri.

Contoh:

const arr = [1, 2, 3];
arr.push(4);  // Menambahkan elemen ke array
console.log(arr);  // Output: [1, 2, 3, 4]
// arr = [5, 6, 7]; // Error: Assignment to constant variable.

6. JS Dates

Tanggal di JavaScript dikelola dengan objek Date. Anda bisa membuat objek tanggal dengan:

  • new Date(): Membuat objek tanggal untuk waktu saat ini.
  • new Date('YYYY-MM-DD'): Membuat objek tanggal untuk tanggal tertentu.
Contoh:

let date = new Date();
console.log(date);  // Output: Tanggal dan waktu saat ini

let specificDate = new Date('2024-12-25');
console.log(specificDate);  // Output: Wed Dec 25 2024

7. JS Date Formats

W3Schools menjelaskan beberapa format tanggal yang dapat digunakan:

  • toISOString(): Format standar internasional.
  • toLocaleString(): Format sesuai dengan pengaturan lokal.
  • toUTCString(): Format tanggal dalam waktu UTC.
Contoh:

let date = new Date();
console.log(date.toISOString());  // Output: 2024-11-24T00:00:00.000Z
console.log(date.toLocaleString());  // Output: format lokal, misalnya "11/24/2024, 12:00:00 AM"

8. JS Date Get Methods

Metode untuk mengambil komponen dari objek Date:

  • getFullYear(): Mengambil tahun.
  • getMonth(): Mengambil bulan (0-11).
  • getDate(): Mengambil tanggal (1-31).
  • getDay(): Mengambil hari dalam seminggu (0-6).
  • getHours(), getMinutes(), getSeconds(): Mengambil jam, menit, dan detik.
Contoh:

let date = new Date();
console.log(date.getFullYear());  // Output: 2024
console.log(date.getMonth());     // Output: 10 (November)
console.log(date.getDate());      // Output: 24

9. JS Date Set Methods

Metode untuk mengubah komponen tanggal:

  • setFullYear(): Mengubah tahun.
  • setMonth(): Mengubah bulan.
  • setDate(): Mengubah tanggal.
  • setHours(), setMinutes(), setSeconds(): Mengubah jam, menit, dan detik.
Contoh:

let date = new Date();
date.setFullYear(2025);
console.log(date.getFullYear());  // Output: 2025

10. JS Math

W3Schools menjelaskan penggunaan objek Math dalam JavaScript:

  • Math.max(): Mengembalikan nilai terbesar dari sekumpulan angka.
  • Math.min(): Mengembalikan nilai terkecil dari sekumpulan angka.
  • Math.random(): Menghasilkan angka acak antara 0 dan 1.
  • Math.abs(): Mengembalikan nilai absolut.
  • Math.round(): Membulatkan angka ke bilangan bulat terdekat.
  • Math.floor(): Membulatkan angka ke bawah.
  • Math.ceil(): Membulatkan angka ke atas.
Contoh:

console.log(Math.max(1, 2, 3));  // Output: 3
console.log(Math.random());      // Output: Angka acak antara 0 dan 1
console.log(Math.abs(-5));       // Output: 5

11. JS Random

Metode Math.random() digunakan untuk menghasilkan angka acak antara 0 dan 1 (tidak termasuk 1).

Contoh:

let randomValue = Math.random();  // menghasilkan angka acak
console.log(randomValue);  // Output: Angka acak antara 0 dan 1

12. JS Booleans

Tipe data Boolean memiliki dua nilai: true dan false.

Contoh:

let isTrue = true;
let isFalse = false;
console.log(isTrue);  // Output: true
console.log(isFalse); // Output: false

13. JS Comparisons

Perbandingan dalam JavaScript menggunakan operator:

  • == (Loose equality): Membandingkan nilai setelah konversi tipe.
  • === (Strict equality): Membandingkan nilai dan tipe data secara ketat.
  • !=, !==: Operasi perbandingan untuk tidak sama.
Contoh:

console.log(5 == '5');  // Output: true (loose equality)
console.log(5 === '5');  // Output: false (strict equality)

14. JS If Else

if-else adalah cara untuk mengontrol alur eksekusi kode berdasarkan kondisi tertentu.

Contoh:

let x = 10;
if (x > 10) {
console.log("X lebih besar dari 10");
} else {
console.log("X kurang dari atau sama dengan 10");
}  // Output: "X kurang dari atau sama dengan 10"

15. JS Switch

Pernyataan switch digunakan untuk membandingkan nilai dengan banyak kondisi.

Contoh:

let x = 2;
switch (x) {
case 1:
console.log("Satu");
break;
case 2:
console.log("Dua");
break;
default:
console.log("Nilai tidak dikenal");
}  // Output: "Dua"

16. JS Loop For

Loop for digunakan untuk iterasi kode sebanyak jumlah yang ditentukan.

Contoh:

for (let i = 0; i < 5; i++) {
console.log(i);  // Output: 0, 1, 2, 3, 4
}
