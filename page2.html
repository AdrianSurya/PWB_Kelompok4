JavaScript Summary

1. JavaScript Objects

Objects are collections of properties (key-value pairs) and methods.

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 30,
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};

console.log(person.fullName()); // Output: John Doe

2. JS Object Properties

Access object properties using dot notation or bracket notation.

const car = { brand: "Toyota", model: "Corolla", year: 2020 };

console.log(car.brand);      // Output: Toyota
console.log(car["model"]);   // Output: Corolla

3. JS Object Methods

Methods are functions stored as object properties.

const calculator = {
  add: function(a, b) {
    return a + b;
  },
  subtract: function(a, b) {
    return a - b;
  }
};

console.log(calculator.add(5, 3));       // Output: 8
console.log(calculator.subtract(5, 3)); // Output: 2

4. JS Object Display

Objects can be displayed using console.log or JSON.stringify.

const user = { name: "Alice", age: 25 };
console.log(user);                     // Output: { name: 'Alice', age: 25 }
console.log(JSON.stringify(user));     // Output: {"name":"Alice","age":25}

5. JS Object Constructors

Constructors are templates for creating objects.

function Person(firstName, lastName) {
  this.firstName = firstName;
  this.lastName = lastName;
  this.fullName = function() {
    return this.firstName + " " + this.lastName;
  };
}

const person1 = new Person("John", "Doe");
console.log(person1.fullName()); // Output: John Doe

6. JS Events

Events are actions or occurrences (e.g., clicks, hovers).

<button onclick="alert('Button clicked!')">Click Me</button>

7. JS Strings

Strings are text enclosed in quotes.

const message = "Hello, World!";
console.log(message.length); // Output: 13

8. JS String Methods

String methods include toUpperCase() and slice().

const text = "JavaScript";
console.log(text.toUpperCase()); // Output: JAVASCRIPT
console.log(text.slice(0, 4));   // Output: Java

9. JS String Search

Search strings using indexOf() or includes().

const text = "Learn JavaScript";
console.log(text.indexOf("Java"));    // Output: 6
console.log(text.includes("Python")); // Output: false

10. JS String Templates

Use template literals with backticks for string interpolation.

const name = "Alice";
const message = `Hello, ${name}!`;
console.log(message); // Output: Hello, Alice!

11. JS Numbers

Numbers are used for integers and floating-point values.

const num = 42;
const pi = 3.14159;
console.log(num, pi);

12. JS BigInt

BigInt is used for very large numbers.

const bigNumber = 123456789012345678901234567890n;
console.log(bigNumber); // Output: 123456789012345678901234567890n

13. JS Number Methods

Examples: toFixed(), toString().

const num = 123.456;
console.log(num.toFixed(2)); // Output: 123.46
console.log(num.toString()); // Output: "123.456"

14. JS Number Properties

Examples: Number.MAX_VALUE, Number.MIN_VALUE.

console.log(Number.MAX_VALUE); // Output: 1.7976931348623157e+308
console.log(Number.MIN_VALUE); // Output: 5e-324

15. JS Arrays

Arrays are collections of elements in a single variable.

const fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Output: Apple

fruits.push("Orange");
console.log(fruits); // Output: ["Apple", "Banana", "Cherry", "Orange"]

fruits.pop();
console.log(fruits); // Output: ["Apple", "Banana", "Cherry"]
