JavaScript Summary

1. JS Objects

JavaScript objects adalah koleksi dari properti yang terkait dengan nilai.

const person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    fullName: function() {
        return this.firstName + " " + this.lastName;
    }
};

console.log(person.fullName()); // Output: John Doe

2. JS Object Properties

Properti objek adalah pasangan key-value. Properti dapat diakses menggunakan notasi titik atau tanda kurung siku.

const car = {
    brand: "Toyota",
    model: "Corolla",
    year: 2020
};

console.log(car.brand);  // Output: Toyota
console.log(car["model"]); // Output: Corolla

3. JS Object Methods

Metode adalah fungsi yang disimpan sebagai properti objek.

const calculator = {
    add: function(a, b) {
        return a + b;
    },
    subtract: function(a, b) {
        return a - b;
    }
};

console.log(calculator.add(5, 3)); // Output: 8
console.log(calculator.subtract(5, 3)); // Output: 2

4. JS Object Display

Objek dapat ditampilkan dengan berbagai cara seperti console.log atau JSON.stringify.

const book = {
    title: "JavaScript Basics",
    author: "Jane Doe",
    year: 2021
};

console.log(book); // Output: {title: "JavaScript Basics", author: "Jane Doe", year: 2021}
console.log(JSON.stringify(book)); // Output: {"title":"JavaScript Basics","author":"Jane Doe","year":2021}

5. JS Object Constructors

Constructor digunakan untuk membuat objek dengan properti yang sama menggunakan fungsi khusus.

function Person(firstName, lastName, age) {
    this.firstName = firstName;
    this.lastName = lastName;
    this.age = age;
}

const person1 = new Person("Alice", "Smith", 25);
console.log(person1.firstName); // Output: Alice

6. JS Events

Event di JavaScript adalah tindakan atau kejadian yang terjadi di halaman web.

7. JS Strings

String adalah urutan karakter yang didefinisikan di dalam tanda kutip.

let greeting = "Hello, World!";
console.log(greeting); // Output: Hello, World!

8. JS String Methods

JavaScript menyediakan berbagai metode untuk memanipulasi string seperti length, toUpperCase(), dan substring().

let text = "JavaScript";
console.log(text.length); // Output: 10
console.log(text.toUpperCase()); // Output: JAVASCRIPT
console.log(text.substring(0, 4)); // Output: Java

9. JS String Search

String dapat dicari menggunakan metode seperti indexOf() atau includes().

let sentence = "Learning JavaScript is fun!";
console.log(sentence.indexOf("JavaScript")); // Output: 9
console.log(sentence.includes("fun")); // Output: true

10. JS String Templates

String templates menggunakan backticks (`) untuk menyisipkan ekspresi menggunakan ${}.

let name = "John";
let age = 30;
console.log(`My name is ${name} and I am ${age} years old.`);
// Output: My name is John and I am 30 years old.

11. JS Numbers

Angka di JavaScript adalah tipe data yang digunakan untuk operasi matematika.

let num = 42;
console.log(num + 8); // Output: 50

12. JS BigInt

BigInt digunakan untuk angka yang sangat besar melebihi batas Number.

let bigNumber = BigInt("123456789012345678901234567890");
console.log(bigNumber); // Output: 123456789012345678901234567890n

13. JS Number Methods

Metode seperti toString() dan toFixed() digunakan untuk memanipulasi angka.

let num = 3.14159;
console.log(num.toFixed(2)); // Output: 3.14

14. JS Number Properties

Properti seperti MAX_VALUE dan MIN_VALUE memberikan informasi tentang batas angka di JavaScript.

console.log(Number.MAX_VALUE); // Output: 1.7976931348623157e+308
console.log(Number.MIN_VALUE); // Output: 5e-324

15. JS Arrays

Array di JavaScript adalah struktur data yang dapat menyimpan banyak nilai dalam satu variabel.

let colors = ["red", "blue", "green"];
console.log(colors[0]); // Output: red
console.log(colors.length); // Output: 3
