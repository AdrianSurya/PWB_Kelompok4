JavaScript Basics

JS Introduction

JavaScript is a versatile language used for web development. It allows you to add interactivity to websites, and is one of the core technologies alongside HTML and CSS.

console.log("Hello, World!");

JS Where To Write Code

You can write JavaScript code in an HTML file using the <script> tag, either inline or in an external file.

<script>
alert("Hello from JavaScript!");
</script>

JS Output

You can display output in JavaScript using console.log(), alert(), or document.write().

console.log("Output in console");
alert("Output in alert box");

JS Statements

JavaScript statements perform an action. Statements end with a semicolon.

let x = 10; // Assignment statement

JS Syntax

JavaScript syntax defines the rules for writing JavaScript code, such as variable declarations and operators.

var name = "John";

JS Comments

Comments are used to explain code. JavaScript supports single-line and multi-line comments.

// Single-line comment
/* Multi-line
   comment */

JS Variables

Variables are used to store data in JavaScript. You can declare variables using var, let, or const.

let name = "Alice";

JS Let

The let keyword is used to declare block-scoped variables.

let greeting = "Hello, World!";
if (true) {
    let greeting = "Hi, there!";
    console.log(greeting); // "Hi, there!"
}
console.log(greeting); // "Hello, World!"

JS Const

The const keyword is used to declare constants whose values cannot be reassigned.

const pi = 3.14;
// pi = 3.14159; // Error: Assignment to constant variable

JS Operators

JavaScript includes various operators for performing calculations, comparisons, and logical operations.

let x = 5;
let y = 10;
let sum = x + y; // Arithmetic operator

JS Arithmetic

Arithmetic operators in JavaScript include +, -, *, /, and %.

let sum = 10 + 5; // 15
let difference = 10 - 5; // 5

JS Assignment

Assignment operators assign values to variables, such as =, +=, -=, etc.

let x = 5;
x += 3; // x = x + 3 -> x = 8

JS Data Types

JavaScript supports various data types like String, Number, Boolean, Object, Array, Null, and Undefined.

let name = "John"; // String
let age = 30; // Number

JS Functions

Functions in JavaScript allow you to group reusable code together and call it when needed.

function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Alice")); // "Hello, Alice!"
